
# EIE Institute Demo Worksheet Generator

This is a demo app to generate English learning worksheets based on teacher-provided articles or links.  
It allows selecting English level (Basic, Intermediate, Advanced) and shows recommended word count ranges.  
The generated worksheet includes:  
- Key vocabulary & phrases  
- Warm-up questions  
- Discussion questions  

This app is built with Streamlit and is meant for internal demo purposes.
